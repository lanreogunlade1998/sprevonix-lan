# Sprevonix Website Source Code

This package contains a complete vanilla HTML/CSS/JS website for the Sprevonix consulting and education platform.

## Included pages
- Home
- About
- Services
- Portfolio / Case Studies
- Contact Us (unified inquiry + booking form)
- Admin dashboard (demo review console)

## Important implementation note
Because this version was constrained to **HTML, CSS, and JavaScript only**, form submissions are stored in **browser localStorage** for demo purposes.

For the full capstone rubric, students should replace localStorage with a real backend + database such as:
- AWS EC2 + Node/Express + MySQL/PostgreSQL
- AWS ECS/Fargate + RDS
- Azure App Service + Azure SQL Database

## Run locally
Open `index.html` directly in a browser or serve the folder with any static server.

## Assets
All illustrations are included as original SVG files in `assets/images`.
